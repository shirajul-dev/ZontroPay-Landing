ডেভেলপার API ডকুমেন্টেশন

সহজেই আপনার অ্যাপ বা ওয়েবসাইটে ZontroPay ইন্টিগ্রেট করুন

বেস ইউআরএল (Base URL)

https://api.zontropay.com/v1

অথেন্টিকেশন (Authentication)

প্রতিটি রিকোয়েস্টে আপনার API Key পাঠাতে হবে।

Authorization: Bearer YOUR_API_KEY

পেমেন্ট তৈরি করুন (Create Payment)

POST /checkout/create

{ "amount": 500, "currency": "BDT", "order_id": "ORD-101", "customer_name": "Abir Hasan", "success_url": "https://yoursite.com/success", "cancel_url": "https://yoursite.com/cancel" }

পেমেন্ট ভেরিফাই করুন (Verify Payment)

POST /checkout/verify

{ "transaction_id": "ZT-998877", "order_id": "ORD-101" }

ওয়েব হুক (Webhook)

পেমেন্ট সফল হলে আমরা আপনার দেওয়া URL-এ একটি POST রিকোয়েস্ট পাঠাবো।

{ "status": "success", "amount": 500, "transaction_id": "ZT-998877", "timestamp": "2026-03-19T10:00:00Z" }
